# **App Name**: InspectAI

## Core Features:

- Responsive UI: Adapts seamlessly to tablet and laptop screen sizes.
- Audio-to-Text QA Input: Allows users to record audio notes during inspections, which are then automatically transcribed by an AI engine.
- Image Attachment: Supports multiple image uploads for each checkpoint to provide visual evidence.
- Admin Checkpoint Customization: Provides an admin interface to set up custom checkpoints for different product categories, types, and clients.
- Structured Report Generation: Generates inspection reports with general comments, conformance statements, next steps, and a finalize button.
- AI-Powered Report Summarization: Leverages AI to analyze checkpoint data and generate short and long summaries for the report conclusion, so the app is able to tool-use to produce accurate conclusions based on all available checkpoint data.
- AI-Suggested Tagging: Automatically suggests relevant tags for non-conformance checkpoints and generates a tag cloud with occurrence counts for better issue tracking, using reasoning.

## Style Guidelines:

- Primary color: Deep, confident blue (#3F51B5) to convey reliability.
- Background color: Soft, desaturated blue (#E8EAF6) for a calm, professional backdrop.
- Accent color: Vivid orange (#FF9800) for CTAs and highlighting key elements.
- Font pairing: 'Space Grotesk' (sans-serif) for headings, and 'Inter' (sans-serif) for body text.
- Use clear, descriptive icons to represent different checkpoint categories and report actions.
- Implement a clean, grid-based layout for easy navigation and content organization.
- Incorporate subtle animations for user feedback and to guide the user through the inspection process.
